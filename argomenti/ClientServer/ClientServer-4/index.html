Web server

Cos'è->Un web server è un'applicazione di un software che, in esecuzione su un server, è in grado di gestire le richieste di trasferimento di pagine web di un client.

La comunicazione avviene tramite protocolli HTTP o HTTPS. Un web server puo' agire come tale in qualsiasi dispositivo in cui c'è un software che puo' funzionare da server web, ma di solito vengono installati in dispositivi hardware ottimizzati per fare cio'. Il software dei server web è accessibile tramite i nomi di dominio dei siti web. Quando un browser richiede una pagina da un server web, il processo deve seguire una serie di passaggi. Inizialmente, un utente deve specificare un URL nella barra degli indirizzi di un browser web. Il browser web otterra' quindi l'indirizzo IP del nome di dominio, traducendo URL tramite DNS. Questo portera' il browser su un server web. Il browser richiedera' quindi il file specifico dal server Web tramite una richiesta HTTP. Il server web rispondera', inviando al browser la pagina richiesta, sempre tramite HTTP. Se la pagina richiesta non esiste o se qualcosa non funziona, il server web rispondera' con un messaggio di errore. Il browser sara' quindi in grado di visualizzare la pagina web.
Web server lato client/server->Il web server, nell'architettura client/server, è il software che implementa la parte server del protocollo HTTP.
Riceve richieste dal client->le elabora->risponde tramite documento HTML. Le caratteristiche da configurare in un web server sono:
-la sicurezza dei documenti offerti;
-i diritti d'accesso dei documenti da parte di classi di utenti;
-modalita' di gestione di file particolari;
-la gestione dello streaming audio e video;
-la configurazione di strumenti di supporto al gestore del sito;
-l'integrazione di programmi o di script lato server da mandare in esecuzione su richiesta del client.
Il web server quando riceve una richiesta dal client puo' interpretarla come una semplice richiesta di invio di pagine HTML statiche all'interno del server o come una richiesta di un file contenente le istruzioni del programma lato server.
