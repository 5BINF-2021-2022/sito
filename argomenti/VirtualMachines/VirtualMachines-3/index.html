Virtual Machines

Quali sono?

Esistono due tipi principali di macchine virtuali: - Si chiama una macchina virtuale di sistema, e viene utilizzata per imitare tutti i processi e i componenti di un computer reale. - Si chiama macchina virtuale di processo, e viene utilizzata per eseguire una sola applicazione e scompare quando viene chiusa.



A seconda delle esigenze dell’utente, le macchine di sistema possono essere configurate per funzionare come se disponessero di una varietà di diversi tipi di hardware.



Le macchine virtuali di processo vengono utilizzate per eseguire un singolo programma in isolamento dal sistema operativo del computer fisico.
