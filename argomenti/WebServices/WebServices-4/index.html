I linguaggi di scripting

I linguaggi di scripting sono basati su un inseme di regole(protocolli), ad esembio che il codice utilizzato per comunicare sia comprensibile da entrambi gli enti,
in modo da stabilire una corretta comunicazione.Il linguaggio utilizzato è formale, quindi non ammette ambiguità.
I linguaggi di programmazzione si dividono in 2 macrocategorie:

Linguaggi a basso livello
- Linguaggio macchina
- Linguaggio assembly


Linguaggi ad alto livello
- Linguaggi compilati
- Linguaggi interpretati

I linguaggi compilati hanno come vantaggio le prestazioni.I linguaggi interpretati hanno come vantaggio la portabilità.Eseguono le istruzioni esattamente come scritte nel codice sorgente.
Un esempio di linguaggio interpretato è lo scripting lato server PHP, che prima viene elaborato da un modulo software installato sul server, e poi viene rimandato indietro in forma di pagina html al client.

Lato client e lato server
I vari linguaggi di scripting vengono suddivisi in base alla macchina che li esegue e sono:
- Linguaggi di scripting lato client: esseguiti dal browser installato nel client.
- Linguaggi di scripting lato server: esseguiti da un interprete (modulo software) installato nel server.

Un esempio di linguaggio di scripting lato server è PHP, che contine un codice dinamico PHP, a differnza del codice statico HTML,
e quindi richiede una traduzione da parte del server da PHP ad HTML per poi rispedire al client il contunuto richiesto.
Un esempio di linguaggio di scripting lato client è il javascript, la differenza principale è che questo linguaggio non viene eseguito dal server ma dal browser dell'utente, cioè dal client.
Javasvript permette lo svolgimento di operazioni che riguardano il sistema dell'utente.
Questo linguaggio a differenza del PHP può essere visualizzato dall'utente e permette di svolgere delle operazioni direttamente sul client,
il linguaggio PHP però può svolgere la funzione di creare il codice della pagina spedita all'utente,
dato che come può creare un codice HTML da spedire all'utente, allo stesso modo può crearlo in javascript.
